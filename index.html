Tech Stack Data Science Project

When I first started programming I had trouble determining which languages and frameworks to learn. So I wrote a script in python to scrape data from the web to figure out which programming technologies were most in demand by employers. As I developed my skillset I learned to store my data in SQL databses, build rest APIs and create a frontend to allow users to interact with my data.
How it works
  • Job requirement data is collected from LinkedIn using a proprietary algorithm.
  • The collected data is checked against a list of programming languages and frameworks to determine which appear most frequently for a given job title.
  • The data is saved to a SQLite databse and accessed through a flask API.
  • The webpage, the database and the flask API are all hosted on a Linux based AWS server.
